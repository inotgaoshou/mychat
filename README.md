# mychat
# 仿微信聊天功能实现,功能包括
1. 语音发送
2. 多图片发送
3. 文字表情发送
4. 聊天记录显示
# 使用到的包括
1. 自定义View
2. 音频播放和录制
3. 自定义图库
4. 定义ImageLoader
5. 学会处理ListView的错位问题
